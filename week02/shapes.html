Playing with shapes

In order to create a circle, use the tag <div>:

<div style="z-index: depth;
      position: relative/absolute;
      left: Xpx (pixels);
      top: Ypx (pixels);
      width: Wpx (pixels);
      height: Hpx (pixels);
      border-radius: 100%;
      background: black;">
</div>

The following shapes were initialised with the following parameter values:

  1. z-index: 5: For a single shape it doesn't have any effect, but for multiple shapes, it defines the relative position in the stack along the z-direction
  2. position: relative: All distances are measured respect to the block immediately before the tag <div>
  3. left: 10px
  4. top: 10px
  5. width: 10px Width of the shape in pixels
  6. height: 10px Height of the shape in pixels. If Height==Width, then a symmetrical shape is created (a circle or square, depending on the value set for border-radius)
  7. border-radius: 50% : A single element makes all four corners circular, while two elements, makes them elliptical
  8. background: black
